# Debug Scripts

Debugging and development scripts (not for production use).

## Scripts

- `debug_mecklenburg.py` - Mecklenburg scraper debugging
- `inspect_mecklenburg_buttons.py` - Inspect Mecklenburg page buttons
- `simple_mecklenburg_test.py` - Simple Mecklenburg test
- `test_golden_cases.py` - Golden test cases
- `test_with_real_scraper.py` - Test with real scraper
- `get_actual_results.py` - Get actual scraping results
- `show_results_proof.py` - Show proof of scraping results

## Usage

These scripts are for development and debugging purposes only.

```bash
# Debug Mecklenburg scraper
poetry run python scripts/debug/debug_mecklenburg.py

# Inspect page structure
poetry run python scripts/debug/inspect_mecklenburg_buttons.py
```

## Note

These scripts may be outdated or specific to development scenarios. Use the organized test scripts in `phase1_1/`, `phase1_2/`, and `phase1_3/` for regular testing.
